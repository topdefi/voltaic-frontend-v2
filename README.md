# VoltaicMining Frontend

## Installation

### Install NodeJS 16
https://github.com/nodesource/distributions/blob/master/README.md#deb


```bash
npm i
```
### Compile

check --base-href in package.json.

```bash
npm run beta
```

### Copy compiled code 

```bash
cp -r dist/voltaicmining-frontend/* target_path
```

### Debug

```bash
npm run start
```
